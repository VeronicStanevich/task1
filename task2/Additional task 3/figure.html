Теги <figure> и <figcaption>.

Элемент <figure>

Элемент <figure> используется в сочетании с <figcaption> и предназначен для разметки диаграмм, иллюстраций, фотографий, примеров кода и других вещей.

Элемент figure представляет собой самодостаточный блок контента с возможным заголовком, на который, как правило, ссылаются из основного потока документа. Этот блок может быть перемещён из основного потока документа без потери общего смысла документа.

Элемент <figcaption>

Элемент figcaption представляет собой заголовок или легенду для figure.

Элемент <figcaption> не обязателен и добавляется до или после содержимого внутри <figure>. Только один <figcaption> может находиться внутри <figure>, при этом допустимо добавлять множество других дочерних элементов (например, <img> или <code>).

Примеры использования.

Элемент figure обычно используется для изображений.

    <figure>

        <img src="orang-utan.jpg" alt="Детёныш орангутанга держится за верёвку">

    </figure>

Несколько изображений внутри figure.

    <figure>
        <img src="kookaburra.jpg" alt="Кукабурра">
        <img src="pelican.jpg" alt="Пеликан на пляже">
        <img src="lorikeet.jpg" alt="Нахально смотрящий многоцветный лорикет">
    </figure>

Пример использования figure для блока с кодом:

<figure>
    <pre>
        <code>
      p {
          color: #333;
          font-family: Helvetica, sans-serif;
          font-size: 1rem;
      }
    </code>
  </pre>
</figure>

Выглядеть будет следующим образом:

Возможно вложение figure в другой тег figure.

<figure role="group">
    <figure>Dog breeds </figure>
    <figure>
        <img src="dog1.jpg" alt="Maltese Terrier">
        <figcaption>Adorable Maltese Terrier</figcaption>
    </figure>
    </figure>
        <img src="dog2.jpg" alt="Black Labrador">
        <figcaption>Cute black labrador</figcaption>
    </figure>
</figure>

Пример использования figcaption.

При использовании figcaption, идеально если он идет первым или последним тегом внутри блока figure.

    <figure>
        <figcaption>Three different breeds of dog.</figcaption>
        <img src="dog1.jpg" alt="Maltese Terrier">
        <img src="dog2.jpg" alt="Black Labrador">
        <img src="dog3.jpg" alt="Golden Retriever">
    </figure>

Или

    <figure>
        <img src="dog1.jpg" alt="Maltese Terrier">
        <img src="dog2.jpg" alt="Black Labrador">
        <img src="dog3.jpg" alt="Golden Retriever">
        <figcaption>Three different breeds of dog.</figcaption>
    <figure>

Или

Можно использовать потоковые элементы в figcaption.

    <figure>
        <img src="dogs.jpg" alt="Group photo of dogs">
        <figcaption>
            <h2>Puppy School</h2>
            <p>Championship Class of 2016</p>
        </figcaption>
    <figure>

Минусы и плюсы использования данных тегов в сравнении с обычной вставкой изображений.

Плюсы

  1. <figure> может быть удалён из документа без ущерба его смыслу
  2. элемент <figure> может содержать несколько дочерних элементов (например, <img> или <code>)
  3. только один элемент <figcaption> может быть помещен в <figure>
  4. возможно размещение нескольких изображений в <figure>, что добавляет удобства

Минусы

  1. если положение содержимого в тексте тесно связано с предыдущим и последующим содержимым, следует использовать более подходящие элементы — например, <div>, старый добрый <img>, <blockquote>
  2. использование элемента <figure>> не всегда целесообразно (например, графический баннер не стоит размечать в <figure>, используйте для этого просто <img>)
